      What is this?

      This service ships machine-readable endpoints (JSON / TXT / INI / CODE) that Skillerious apps call to check for updates.

      • No push installs - the site never installs software for you.
      • Public metadata - versions, release links, integer codes.
      • Client-driven - each app decides what to do with the data.
